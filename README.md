### ⚠️ 安装注意事项（重要）

**Windows 用户**

* 安装程序请右键选择「以管理员身份运行」
* 如提示不安全，点击「更多信息 → 仍要运行」

**Android 用户**

* 出现安全或备案提示，选择继续安装即可
* 首次连接需确认 VPN 权限

**macOS 用户**

* Apple 芯片（M 系列）请选择 `arm64.pkg`
* Intel 芯片请选择不带 `arm64` 后缀的安装包
* 如被系统拦截，请前往「系统设置 → 隐私与安全」允许打开
* 并授予 **完全磁盘访问权限** 以启用 TUN 模式
